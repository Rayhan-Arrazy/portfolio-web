# Rayhan Arrazy - Developer Portfolio

A modern, fast, and beautiful portfolio landing page built with Next.js App Router to showcase skills, projects, and contact information.

## Tech Stack
- **Frontend**: Next.js 14 (App Router), React, TypeScript, SCSS (Vanilla CSS variables & Modules), Framer Motion, Lucide React.
- **Backend API**: Next.js API Routes.
- **Email Delivery**: Nodemailer with Ethereal Email (Mock SMTP for testing).
- **AI Integration**: Gemini API via `@google/genai`.

## Features
1. **Dynamic UI**: Glassmorphism, smooth scroll, dark mode aesthetics, and micro-animations using Framer Motion.
2. **Projects & Experience**: Showcases full-stack capabilities, drawing from real GitHub data.
3. **Contact Form**: Implements the full cycle (Frontend → API → Nodemailer). A confirmation email is generated and sent to the user, and an alert is sent to the site owner. It uses **Ethereal Email** to output a test preview URL in the console—meaning it works 100% out of the box without real SMTP credentials.
4. **AI Assistant**: Features a built-in AI chat widget powered by Gemini API to answer questions about the portfolio.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Variables**
   Create a `.env.local` file in the root of the project to enable the AI features:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
   *Note: If no API key is provided, the AI Assistant will elegantly gracefully fallback to a pre-defined informational response.*

3. **Run the Development Server**
   ```bash
   npm run dev
   ```

4. **Testing the Contact Form**
   Submit the contact form on the frontend. Check your terminal where `npm run dev` is running. You will see a clickable Ethereal Email preview URL generated by Nodemailer.

## AI Implementation Details
- **What AI tools were used**: The new `@google/genai` SDK is integrated via the Next.js App Router (`/api/chat`).
- **What was done using AI**: The chatbot acts as an interactive agent that answers queries about Rayhan's skills.
- **What had to be fixed manually**: Graceful fallbacks and UI loading states were implemented manually to ensure the page remains beautiful and functional even if the API key is absent. Error boundaries and loading spin states are entirely custom-built.
